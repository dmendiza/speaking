Models

  • Each model is a Python class that subclasses django.db.models.Model
  • Each class attribute of the model represents a database field
     1 from django.db import models
 2 
 3 # Create your models here.
 4 
 5 class Author(models.Model):
 6     first_name = models.CharField(max_length=256)
 7     last_name = models.CharField(max_length=256)
 8 
 9 class Book(models.Model):
10     author = models.ForeignKey(Author)
11     title = models.CharField(max_length=256)
12     release_date = models.DateField(null=True)
13     isbn = models.CharField(max_length=13)
14     list_price = models.DecimalField(max_digits=8, decimal_places=2)
15     on_hand = models.IntegerField(default=0)

  • You can execute raw SQL if you need to

Security - Cross Site Request Forgery

  • Easy to use
  • Not on by default

  • Ensure middleware is enabled

    1 'django.middleware.csrf.CsrfViewMiddleware'

  • In any template that uses a POST form use

    1 <form action="." method="post">
2     {% csrf_token %}

  • Use render which will us a RequestContext for you

    1 render(request, template)
